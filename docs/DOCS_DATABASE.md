# RAG Dokumentum: Adatbázis

A rendszer adatbázis-kezelője a PostgreSQL 17. A séma relációs, normalizált, de a PostgreSQL JSONB adattípusát használja a rugalmas, nem strukturált adatok (pl. termékmódosítók, fordítások, NTAK adatok) hatékony tárolására. A teljes adatbázis séma a `DATABASE_SCHEMA.md` fájlban található.
